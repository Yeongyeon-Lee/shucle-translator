# shucle-translator

## Overview

셔클 모빌리티 플랫폼 전용 번역 스킬. 한국어 텍스트를 5개 언어로 번역한다.
- shucle-terms 용어집을 참조해 셔클 도메인 용어를 정확하게 처리
- i18n-translator 원칙에 따라 각 언어의 자연스러운 UX 톤 적용
- 서비스명·브랜드명(Shucle, DRT, 똑버스 등)은 번역하지 않고 원문 유지

## Usage

```
/shucle-translator
해지 취소하기
```

```
/shucle-translator
라이더가 호출을 완료했습니다. 드라이버앱에서 배차 현황을 확인해 주세요.
```

## 실행 절차

1. `.claude/skills/shucle-terms/SKILL.md`를 읽어 원문에 포함된 셔클 도메인 용어를 파악한다
2. `.claude/skills/i18n-translator/SKILL.md`의 번역 원칙과 언어별 톤 가이드를 적용한다
3. 아래 규칙에 따라 5개 언어로 번역한다
4. 아래 출력 형식으로 결과를 제공한다

## 번역 규칙

### 핵심 원칙
- **직역 금지**. 각 언어권 원어민이 자국 모빌리티 서비스를 사용하는 것처럼 자연스럽게 번역
- Uber, Bolt, Grab, FREE NOW 등 글로벌 모빌리티 서비스의 표현과 UX 패턴 참고
- **서비스명·브랜드명·고유명사는 원문 그대로** 유지 (Shucle, DRT, 똑버스, MaaS 등)
- 변수/플레이스홀더(`{{variable}}`, `{0}`, `%s` 등)는 절대 번역하지 않고 그대로 유지
- UI 텍스트는 원문 길이의 120% 이내로 유지

### 언어별 톤

| 언어 | 톤 | 참고 서비스 |
|------|-----|------------|
| **영어 (en)** | 캐주얼하고 직접적. 명령형 허용. "Please kindly" 같은 과도한 정중함 금지 | Uber, Lyft |
| **헝가리어 (hu)** | tegezés("te" 비격식) 사용. magázás("Ön" 격식) 절대 금지 | Bolt, Wolt |
| **일본어 (ja)** | です/ます체. 정중하고 자연스러운 서비스 어투 | Uber Japan, GO |
| **중국어 간체 (zh_CN)** | 자연스러운 중국 앱 서비스 표현 | 滴滴, 高德 |
| **중국어 번체 (zh_TW)** | 대만 앱 서비스 표현. 간체와 다른 어휘 사용 | Uber Taiwan |

### UI 요소별 톤

| 요소 | 톤 |
|------|-----|
| 버튼/CTA | 간결한 행동 유도형 (2~3단어 이내) |
| 안내 문구 | 정중하고 명확한 서술형 |
| 에러 메시지 | 정중하되 명확하게. 사용자 탓 느낌 금지 |
| Placeholder | 부드러운 안내형 |
